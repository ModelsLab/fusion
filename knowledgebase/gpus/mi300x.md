## Specs
- Architecture: CDNA 3 (chiplet design, 8 XCDs)
- Compute Units: 304, Matrix Cores: 1216
- FP16: 1307 TFLOPS, FP8: 2615 TFLOPS, BF16: 1307 TFLOPS
- Memory: 192 GB HBM3, 5.3 TB/s bandwidth
- Infinity Cache: 256 MB
- Interconnect: 7x Infinity Fabric links (896 GB/s)
- TDP: 750W

## Key Advantage
Can fit LLaMA-70B in FP16 on a single chip (192 GB > 140 GB weight size).
The H100 (80 GB) cannot do this without quantization or multi-GPU.

## Software
- ROCm 6.x with HIP runtime
- PyTorch ROCm builds (torch.cuda works via HIP)
- vLLM and SGLang support MI300X
- Composable Kernel (CK) library for optimized GEMM
- Flash Attention for ROCm available
